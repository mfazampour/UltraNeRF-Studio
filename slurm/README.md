# Slurm

This folder contains cluster-job helpers for running experiments under Slurm.

- `job_test_slurm.sh`
  Basic environment / scheduler test job.

- `job_generate_perturbed_poses.sh`
  Batch job for generating perturbed pose data.

- `slurm_tester.py`
  Simple PyTorch availability test for cluster environments.

Treat these as deployment helpers rather than core algorithm code.
